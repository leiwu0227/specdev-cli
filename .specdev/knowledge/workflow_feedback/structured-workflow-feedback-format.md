# Structured Workflow Feedback Format

Status: resolved
Type: improvement
Severity: moderate
First seen: 2026-05-10, 00016_refactor_distill-workflow
Last seen: 2026-05-10, 00016_refactor_distill-workflow
Assignments observed: 00016_refactor_distill-workflow

## Observation
- Workflow-related improvements were previously accumulated through free-form `capture/workflow-diff.md` reflections and optional `knowledge/workflow_feedback/` notes without a required durable note format.

## Impact
- Agents could create duplicate or inconsistent workflow feedback notes, and repeated issues had no clear status, severity, mitigation, or promotion path into future assignments.

## Current Mitigation
- New installs receive `_templates/workflow_feedback_note.md`, and Knowledge Capture guidance now tells agents to classify workflow observations, search existing feedback, update repeated issues, and use structured fields.
