# Bounded Working Memory Implementation Plan

> **For agent:** Implement this plan task-by-task using TDD discipline.

**Goal:** Add a deterministic, bounded `.specdev/project_notes/working_memory.md` artifact that agents can refresh and read cheaply.

**Architecture:** Add a `memory` command using the same command-family pattern as `skills`: register `memory` in `dispatch.js`, then route `refresh` inside `src/commands/memory.js`. Put generation logic in a reusable utility so command output, JSON metadata, and tests share one implementation. Keep `distill done` JSON-only by adding a `memory_hint` field.

**Tech Stack:** Node.js ESM CLI, `fs-extra`, existing assignment scanning/state utilities, plain Node.js tests.

**Execution Mode:** inline

**Step 3: Write minimal implementation**
Create `src/commands/memory.js` with:

```js
import { join } from 'path'
import fse from 'fs-extra'
import { resolveTargetDir, requireSpecdevDirectory } from '../utils/command-context.js'

export async function memoryCommand(positionalArgs = [], flags = {}) {
  const subcommand = positionalArgs[0] || 'refresh'
  if (subcommand === 'refresh') return memoryRefreshCommand(flags)
  console.error(`Unknown memory subcommand: ${subcommand}`)
  console.log('Usage: specdev memory refresh')
  process.exitCode = 1
}

async function memoryRefreshCommand(flags = {}) {
  const targetDir = resolveTargetDir(flags)
  const specdevPath = join(targetDir, '.specdev')
  await requireSpecdevDirectory(specdevPath)
  const outputPath = join(specdevPath, 'project_notes', 'working_memory.md')
  const content = '# Working Memory\n\nGenerated by `specdev memory refresh`. Do not edit by hand.\n'
  await fse.ensureDir(join(specdevPath, 'project_notes'))
  await fse.writeFile(outputPath, content, 'utf-8')
  if (flags.json) {
    console.log(JSON.stringify({
      command: 'memory refresh',
      version: 1,
      status: 'ok',
      output_path: '.specdev/project_notes/working_memory.md',
    }, null, 2))
    return
  }
  console.log('Working memory refreshed: .specdev/project_notes/working_memory.md')
}
```

```js

```
