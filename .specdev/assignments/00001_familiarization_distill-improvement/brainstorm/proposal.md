# Investigation: SpecDev Distill Function

## What needs understanding
The `specdev distill` command family — how it works today, whether it achieves its intended purpose, and if improvements are needed.

## Why
The user wants to evaluate the distill system's effectiveness and discuss potential improvements.

## Learning Objectives
1. How does the distill pipeline work end-to-end (capture → distill → mark-processed)?
2. What heuristics exist and are they useful?
3. Is the agent-driven JSON approach working as intended?
4. Are there gaps or friction points?
5. Should anything be added, changed, or removed?
